# نظام التعلم الذاتي الشهري

## التنفيذ

في بداية كل شهر (أو عند طلب المستخدم "تحديث الاستراتيجيات" أو "self-improve")، نفذ الخطوات التالية:

### الخطوة 1: بحث عميق في أحدث التوجهات

ابحث في الإنترنت عن:
1. "digital marketing trends [current year] Middle East" 
2. "social media algorithm changes [current month] [current year]"
3. "best performing ad formats [current year]"
4. "social media statistics Arab countries [current year]"
5. "[platform name] new features [current year]"

### الخطوة 2: تحليل التغييرات

قارن النتائج مع المعلومات الحالية في Skills:
- هل تغيرت أوقات الذروة لأي منصة؟
- هل هناك ميزات جديدة في أي منصة؟
- هل تغيرت معايير الأداء (benchmarks)؟
- هل هناك توجهات جديدة في المحتوى؟
- هل تغيرت أسعار الإعلانات؟

### الخطوة 3: تحديث Skills

اقترح تحديثات محددة لكل Skill متأثر:
- marketing-strategy: تحديث الإحصائيات والمعايير
- social-media-expert: تحديث المواصفات وأفضل الممارسات
- ad-campaign-optimizer: تحديث معايير الأداء والقواعد
- content-creator-ar: تحديث أنماط المحتوى الرائجة
- report-designer: تحديث القوالب إن لزم

### الخطوة 4: إنشاء تقرير التحديث

أنشئ تقريراً يشمل:
```
## تقرير التعلم الذاتي - [الشهر/السنة]

### التغييرات المكتشفة:
1. [تغيير 1]
2. [تغيير 2]

### التحديثات المطبقة:
1. [تحديث 1]
2. [تحديث 2]

### التوصيات الجديدة:
1. [توصية 1]
2. [توصية 2]

### التأثير المتوقع:
- [أثر 1]
```

### الخطوة 5: تحديث platform_statistics

حدّث جدول platform_statistics في Supabase بأحدث البيانات المتاحة لكل دولة.
