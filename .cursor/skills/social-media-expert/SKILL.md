# Social Media Platform Expert

## Platform Specifications & Best Practices

### Facebook (Meta)
- **API:** Graph API v24.0, Marketing API v24.0
- **Post Types:** Text, Image, Video, Link, Carousel, Reel, Story, Live
- **Image Specs:** 1200x630 (link), 1080x1080 (square), 1080x1350 (portrait)
- **Video:** Max 240 min, 4GB, recommended < 3 min
- **Reels:** 15-90 sec, 9:16 aspect ratio
- **Best Practices:**
  - Native video outperforms links
  - Questions and polls drive engagement
  - Facebook Groups for community building
  - Reels get 2-3x more reach than static posts

### Instagram (Meta)
- **API:** Instagram Graph API (via Meta Graph API v24)
- **Post Types:** Feed Post, Story, Reel, Carousel, Live, Guide
- **Feed Image:** 1080x1080 (square), 1080x1350 (portrait - recommended)
- **Story/Reel:** 1080x1920 (9:16)
- **Carousel:** Up to 10 slides
- **Reels:** 15-90 sec, highest reach format
- **Best Practices:**
  - Reels are #1 for reach and discovery
  - Carousel posts get highest saves
  - Use 20-30 hashtags (mix of sizes)
  - Stories for daily engagement
  - Alt text for accessibility and SEO

### TikTok
- **API:** Content Posting API, oauth-based
- **Video:** 15 sec - 10 min, 9:16, max 10GB
- **Best Practices:**
  - Hook viewers in first 3 seconds
  - Use trending sounds
  - Native-looking content outperforms polished
  - 3-5 hashtags maximum
  - Post 1-3 times daily for growth
  - Duets and Stitches for engagement

### Snapchat
- **API:** Snapchat Marketing API
- **Content:** Snaps, Stories, Spotlight, AR Lenses
- **Specs:** 1080x1920, 3-60 sec video
- **Best Practices:**
  - Dominant in Gulf (especially Saudi, UAE, Kuwait)
  - AR filters for brand engagement
  - Behind-the-scenes content performs well
  - Spotlight for viral potential

### X (Twitter)
- **API:** X API v2
- **Post Types:** Text (280 chars), Image, Video, Poll, Thread, Space
- **Image:** 1600x900 (16:9), up to 4 images
- **Video:** Max 2:20, 512MB
- **Best Practices:**
  - Threads for storytelling
  - Engage in trending topics
  - Quick responses to events
  - Polls for engagement
  - Spaces for thought leadership

### LinkedIn
- **API:** LinkedIn Marketing API
- **Post Types:** Text, Image, Video, Document, Article, Newsletter, Poll
- **Best Practices:**
  - Long-form text posts with line breaks perform best
  - Document/carousel posts for highest engagement
  - Professional tone, value-driven content
  - Best for B2B, recruitment, industry news
  - Post during business hours (Sun-Thu in Arab region)

### YouTube
- **API:** YouTube Data API v3
- **Content:** Videos, Shorts, Live, Community Posts
- **Shorts:** < 60 sec, 9:16
- **Best Practices:**
  - SEO-optimized titles and descriptions
  - Custom thumbnails (1280x720)
  - Shorts for discovery, long-form for retention
  - Chapters for long videos
  - End screens and cards for CTAs

## Hashtag Strategy

- **Instagram:** 20-30 hashtags (5 large, 10 medium, 10 small, 5 niche)
- **TikTok:** 3-5 hashtags (trending + niche)
- **X:** 1-3 hashtags maximum
- **LinkedIn:** 3-5 hashtags
- **Facebook:** 1-3 or none

## Content Calendar Integration

When creating content calendars, include:
1. Date and time (localized to target timezone)
2. Platform(s)
3. Content type (post, story, reel, etc.)
4. Copy text (primary + alt versions for A/B)
5. Visual description or asset reference
6. Hashtags
7. CTA (Call to Action)
8. Campaign/theme tag
9. Status (draft → approved → scheduled → published)
