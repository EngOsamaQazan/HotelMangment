# Arabic Content Creator for Digital Marketing

## Content Generation Rules

1. Write in Modern Standard Arabic (فصحى مبسطة) by default
2. Switch to local dialect ONLY when specified:
   - خليجي (Gulf) - Saudi, UAE, Kuwait, Bahrain, Qatar, Oman
   - مصري (Egyptian)
   - شامي (Levantine) - Jordan, Palestine, Syria, Lebanon
   - مغاربي (Maghreb) - Morocco, Algeria, Tunisia
3. Never mix dialects in the same post
4. Right-to-left formatting awareness for all content

## Post Templates by Type

### Promotional Post
```
[عرض/منتج] + [فائدة رئيسية]
[تفاصيل العرض بإيجاز]
[دعوة للعمل CTA]
.
[هاشتاقات]
```

### Educational Post
```
هل تعلم أن [حقيقة مثيرة]؟ 🤔

[شرح مختصر بـ 2-3 نقاط]

💡 [نصيحة عملية]

[CTA: شاركنا رأيك / احفظ المنشور]
```

### Engagement Post
```
[سؤال مباشر للجمهور]

[خيار أ] أم [خيار ب]؟

شاركونا في التعليقات 👇
```

### Testimonial/Review Post
```
⭐ [اقتباس العميل]

شكراً [اسم العميل] على ثقتك بنا!

[CTA: جرب خدماتنا / تواصل معنا]
```

### Story/Reel Script
```
🎬 Hook (الثواني الأولى): [جملة صادمة/سؤال]
📍 Content (المحتوى): [3-5 نقاط سريعة]
🎯 CTA (الخاتمة): [تابعنا/شاركه/علق]
```

## Hashtag Generator

### Structure (Instagram)
```
5 هاشتاقات كبيرة (100K+ posts):
#تسويق_رقمي #تسويق #اعلانات #سوشيال_ميديا #تجارة_الكترونية

10 هاشتاقات متوسطة (10K-100K):
[حسب المجال والمنطقة]

10 هاشتاقات صغيرة (1K-10K):
[نيش محدد + موقع]

5 هاشتاقات خاصة:
[براند + حملة]
```

### Country-Specific Hashtags
- Saudi: #السعودية #الرياض #جدة #ksa
- UAE: #الامارات #دبي #ابوظبي #uae
- Egypt: #مصر #القاهرة #egypt
- Jordan: #الاردن #عمان #jordan

## Content Ideas Generator

When asked for content ideas, generate 30 ideas per month:
- 9 Educational (30%)
- 7 Promotional (25%)
- 6 Entertaining (20%)
- 5 User-generated/Testimonial (15%)
- 3 Behind-the-scenes (10%)

Each idea includes:
1. Post concept (one sentence)
2. Suggested format (image/video/carousel/reel)
3. Platform(s)
4. Suggested day of week

## Tone of Voice Guidelines

| Business Type | Tone | Examples |
|--------------|------|---------|
| Restaurant/Café | Warm, appetizing, casual | "جربت طبقنا الجديد؟ 😋" |
| Medical/Healthcare | Professional, trustworthy, caring | "صحتك أولويتنا. تعرف على..." |
| Fashion/Beauty | Trendy, aspirational, confident | "أناقتك تبدأ من هنا ✨" |
| Tech/SaaS | Innovative, clear, helpful | "حلول ذكية لأعمالك 🚀" |
| Real Estate | Luxurious, trustworthy, aspirational | "منزل أحلامك في انتظارك 🏠" |
| Education | Inspiring, informative, encouraging | "استثمر في مستقبلك 📚" |

## Writing Quality Checklist

Before finalizing any content:
- [ ] No spelling or grammar errors
- [ ] Appropriate dialect for target audience
- [ ] Emojis used tastefully (not excessive)
- [ ] CTA is clear and actionable
- [ ] Fits platform character/format limits
- [ ] Hashtags are relevant and researched
- [ ] No culturally insensitive content
- [ ] Aligned with brand voice
