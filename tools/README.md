# أدوات التشغيل (Windows)

مُشغِّلات سريعة للعمليات اليومية. شغّل أيّاً منها بدبل كليك من مستكشف الملفات.

| الملف | الوظيفة | يتطلّب Admin |
|---|---|---|
| [`setup-prod-db.cmd`](./setup-prod-db.cmd) | فتح نفق SSH إلى قاعدة بيانات الإنتاج (للتطوير عبر النفق) | لا |
| [`migrate-to-local-db.cmd`](./migrate-to-local-db.cmd) | نسخ بيانات الإنتاج مرّة واحدة إلى PostgreSQL محلية + إعادة توجيه `.env` للمحلي | نعم |
| [`install-as-service.cmd`](./install-as-service.cmd) | تسجيل `hotel-app` كمهمة مجدولة تبدأ مخفية عند تسجيل الدخول | نعم |

هذه المُشغِّلات مجرّد أغلفة بسيطة حول سكربتات PowerShell في
[`../scripts/`](../scripts/) — المنطق كامل يوجد هناك.
